Application Manual

This manual provides instructions on how to set up, run, and use the application. All command examples are provided as plain text within code blocks, not as images.

1. Prerequisites

Before running the application, ensure you have the following installed and configured:

  • Python 3.x installed and added to your system's PATH.
  • The whisper.cpp project successfully compiled. The application depends on the executables (stream.exe, main.exe) being in the build/bin/Release directory.
  • The required GGML models (e.g., ggml-base.en.bin) downloaded and placed in the models directory.

2. Installation

You need to install the required Python libraries. These are listed in the requirements.txt file.

Open a terminal or command prompt, navigate to the TempoReal_Karaoke directory, and run the following command:

pip install -r requirements.txt

3. Running the Application

Once the dependencies are installed, you can start the web server.

From the TempoReal_Karaoke directory, run this command:

python app.py

After running the command, you should see output indicating that the server is running, similar to this:

 * Serving Flask app 'app'
 * Running on http://127.0.0.1:5000 (Press CTRL+C to quit)

You can now access the application by opening your web browser and navigating to http://127.0.0.1:5000.

4. Using the Application

Tab 1: Real-time STT

This tab provides real-time speech-to-text from your microphone.

  1. Click the Start button. Your browser may ask for permission to access your microphone.
  2. Begin speaking. The transcribed text will appear in the text box below.
  3. Click the Stop button to end the stream.

Tab 2: Audio Transcription

This tab allows you to transcribe an entire audio file.

  1. Click "Choose File" and select an audio file from your computer.
  2. Click the Transcribe Audio button.
  3. The transcription will appear in the result box below.

Tab 3: Generate Video

This tab generates a new video file with burned-in subtitles.

  1. Click "Choose File" and select a video file.
  2. Click the Generate Video button.
  3. The process may take some time depending on the video length.
  4. Once complete, a download link for the new video will appear.

Note: The video generation process is resource-intensive. Processing long videos may be slow and consume significant CPU resources.
